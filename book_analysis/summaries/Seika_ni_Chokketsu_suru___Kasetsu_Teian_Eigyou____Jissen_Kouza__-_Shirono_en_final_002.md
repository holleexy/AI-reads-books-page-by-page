## 仮説提案営業とは何か
**「仮説提案営業」**とは、顧客に明確なニーズがない状態でも商談を確実に前進させることができる営業手法です。事前調査に基づき、限られた情報から顧客の課題と解決策の「仮説」を立てて提案ストーリーを構築し、初回商談から個別の提案資料を持って臨みます。

### 仮説思考と網羅思考
- ***網羅思考***: すべての事象やデータを抽出・分析してから結論を出す手法。時間がかかりすぎ、営業の現場には不向き。
- ***仮説思考***: ボストン・コンサルティング・グループ（BCG）が提唱したフレームワーク。十分な情報が揃っていない段階で「仮の結論」を立て、実行・検証・修正を繰り返す手法。営業活動におけるスピードと効率を劇的に高めます。

### 仮説提案営業の基本プロセス
1. **仮説を立てる**
2. **提案資料の準備を行う**
3. **商談を組み立てる**
4. **受注獲得する**

## 仮説立案のステップ
営業において立案すべき仮説は、主に「問題仮説」と「解決仮説」の2つです。
- **問題仮説**: 顧客が抱えているであろう課題の仮定（例：「業績が落ちているのは○○が原因だろう」）
- **解決仮説**: 問題を解決するための具体的な対策の仮定（例：「××製品を導入すべきだ」）

### So what?（だから何なのか？）による深掘り
課題の根本原因や与えるインパクトを論理的に繋げるために、「So what?」という自問自答を繰り返す手法を用います。

**【具体例：脆弱性対策製品の提案】**
- 事実：システムに脆弱性がある
- **So what?**：攻撃ターゲットになりやすい
- **So what?**：機密情報が漏洩するリスクがある
- **So what?**：数億円規模の損失や社会的信用の失墜に繋がる
- **解決策**：したがって、自動でパッチを適用する製品Zが必要である

### 仮説レベルの4段階
顧客の具体情况に合わせたストーリーを作るために、仮説は4つのレベルに分けられます。レベルが上がるほど案件化率が向上します。
1. **レベル①（エリア特化）**: 「日本の企業の課題」

4. **レベル④（企業特化）**: 「その顧客ならではの課題」（最も説得力が高く共感を得やすい）

## 提案資料の作成
オンライン商談において、提案資料の「質」が商談の成否を大きく左右します。

### 資料作成の3つのポイント
1. **商談ゴールを明確にする**: 攻めの営業の初回商談におけるゴールは、「製品の機能を理解してもらうこと」ではなく「自社製品導入の必要性を理解してもらうこと」です。
2. **情報を詰め込みすぎない**: 「思考と情報のパラドクス」（情報量が増えるほど相手の思考量が減る現象）を避け、顧客に思考の余白を残すよう情報を厳選します。
3. **顧客が知りたい情報を入れる**: Why? × 4W + 2H のフレームワークを意識し、「どうやって（How）」ではなく「どうなるか（What will happen）」を中心に構成します。

### スライド作成の7つのチェックポイント
1. 色が多過ぎないか（3色以内を推奨し、ルール化する）
2. 余白は十分あるか（圧迫感を与えない）
3. 1スライドに複数メッセージがないか（伝えたいことは1スライド1つ）
4. 言葉のダブりがないか（冗長な表現を避ける）
5. 表が見やすいか（シマウマ表を避け、強調箇所にのみ色を付ける）
6. グラフが見やすいか（余計な線を削る）
7. ラインが揃っているか（配置を整え、手抜き感を排除する）

### 導入事例（ケーススタディ）の活用
事例は、顧客と同業種・同規模のものを優先して提示します。新製品で導入実績がない場合は、以下のように段階的にアピールします。
- **検討中事例** → **検証事例** → **導入事例**
- 「すでに10社が検討中で、うち5社が検証済み。3社は御社と同業界です」と具体的人数を提示。

## 商談の進め方
### 初回商談のタイムスケジュール（60分の場合）
- **オープニング（5分）**: アイスブレイク、自己紹介、目的説明
- **ボディ（45分以内）**: プレゼン＆ヒアリング、質疑応答
- **クロージング（10分）**: 次回のアクション確認、次回日程確定

### オープニングと信頼構築
- **雑談質問**: 事前調査に基づき「創業周年おめでとうございます」など、相手への関心を示して警戒心を解きます。
- **自己紹介**: 「100社以上のコスト削減を支援してきたノウハウを活かし、御社の成長に貢献したい」といった、過去の実績と顧客へのベネフィットを含めます。
- **目的説明**: 「事前に調査し、御社の課題の仮説を立ててきました」とアピールし、「有益でなければ次のステップに進まなくて結構です」と逃げ道を用意することで、顧客の心理的ハードルを下げます。

### ヒアリング（ボディ）
- **仮説の提示**: 「同業他社でこのような課題がよく聞かれますが、御社の実際はいかがですか？」と問いかけます。
- **返報性の法則**: こちらが事前調査情報を提供することで、顧客も情報を出しやすくなります。
- **5W2Hの活用**: 顧客が話し始めたら、ただ聞くのではなく、「いつから？」「どのような対処を？」と核心を突く質問を重ねます。
- **仮説が外れた場合**: 決して無理に製品を売り込まず、プレゼンを中止して新たな仮説を立て、ヒアリングに徹します。

### Q&Aのコツ
- **質問のタイミング**: プレゼン後に「ご質問はありますか？」と聞くのではなく、10分おきに「ここまでで気になったことはありますか？」と心理的ハードルを下げて促します。
- **即答できない場合**: 「わかりません」と言うのではなく、「おそらく可能ですが、エンジニアに確認して確実にお答えします」と前向きに返答します。
- **質問に質問で返す**: 「それは必須要件（MUST）ですか？実現すればいつ頃導入されますか？」と、顧客の確度と熱量を確認します。

### 選択肢クロージング（松竹梅クロージング）
初回商談で「いかがでしょうか？」や「ご検討ください」と即決を迫るのはNGです。心理学を応用し、「今後の進め方として、A（松：詳細検証）、B（竹：小規模検証）、C（梅：情報提供のみ）の3つのパターンを用意しました。どれがよろしいですか？」と選択肢を提示し、自然に次回ステップへ誘導します。

### 商談後のフォローアップ
商談終了後、記憶が薄れる前に（1時間以内に）**議事メモ（ミーティングminutes）**をメールで送信します。エビングハウスの忘却曲線に基づき、人間の記憶は1時間後に50%消失するため、スピーディーな対応が信頼獲得に直結します。また、長期案件の放置を防ぐため、スケジューラにリマインドを設定しておくことが重要です。

## パートナーセールス（代理店・販売店営業）
日本のIT市場では、メーカーが直接顧客に売るのではなく、SIerや代理店（パートナー）を通じて販売する間接販売が主流です。

### パートナーセールスの5つのステップ
1. **パートナー分析**: 企業のビジネス状況や戦略を調査しアカウントプランを作成。
2. **協業提案**: 自社・顧客・パートナーの「Win-Win-Win」になるストーリーを提案。
3. **調整・交渉**: 仕切り、リベート、サポートフローなどを合意。
4. **案件創出**: 勉強会や営業同行を通じて見込み案件を獲得。
5. **継続的な受注創出**: イノベーター理論を応用した展開と定例会でのPDCA回し。

### 協業提案のコツ
- **共存戦略（棲み分け）**: 既に競合他社がいる場合、他社を排除するのではなく、「大企業向けは競合、中小企業向けは自社」と棲み分けを提案すると受け入れられやすい。
- **役職別のアピール**: 経営層には「中長期的な事業戦略の推進」、管理者には「半年で6000万円の売上増」、現場には「受注サイクルが平均2ヶ月で今期の足しになる」など、相手の関心事に合わせてメッセージを変える。

### 効果的な勉強会の進め方
パートナー企業の営業担当者は「情報過多による思考停止」と「現状維持バイアス」に陥りやすいため、製品機能を詳細に教え込む勉強会は失敗します。
- **ゴールの設定**: 「製品を覚えてもらうこと」ではなく、「提案できそうな顧客を紹介してもらうこと」に設定する。
- **小人数制と短時間**: 課やチーム単位の10〜20分程度に留める。
- **切り口**: 「今提案すれば今期中に受注できる」「〇〇の課題を持つ中小企業に刺さる」と相手の目線に徹底的に立つ。

### イノベーター理論を応用した展開
組織に新製品を普及させるには、以下のステップを踏みます。
1. **アーリーアダプター（約1割）を見つける**: 提案に前向きな担当者を見つけ、徹底的にサポートして成功体験（受注）を作る。
2. **水平展開**: その成功事例（金額、期間、営業トークなど）を定例会などで共有し、「アーリーマジョリティ（多数派）」が「自分もやりたい」と思うように仕向ける。

## まとめと営業哲学
### 顧客の心を動かす「6つの心理」（ロバート・B・チャルディーニ提唱）
1. **返報性**: 個別最適化された仮説提案を行うことで、「ここまで考えてくれたなら真剣に聞こう」と思わせる。
2. **一貫性**: 議事メモで合意事項を残し、約束通りに行動させる。
3. **社会的証明**: 「同規模の同業他社でも導入が進んでいます」と安心感を与える。
4. **権威**: 「〇〇省のガイドラインで推奨されています」と客観的な権威を示す。
5. **好意**: アイスブレイクや熱意ある自己紹介で好印象を与える。
6. **希少性**: 「無償トライアルは残り2社限定です」と稀缺性を強調する。

### 「個人（あなた）」として選ばれる営業へ
情報が溢れ、製品の同質化が進む現代において、「何を買うか」よりも「誰から買うか」が重要視されています。属する企業のブランドや製品のスペックに頼るのではなく、入念な事前準備に基づく「仮説提案」を通して、「あなたが提案してくれたから導入した」「あなただから一緒にビジネスをしたい」と顧客から選ばれる営業パーソン（個のブランド）を目指すことが、仮説提案営業の究極の目標です。
