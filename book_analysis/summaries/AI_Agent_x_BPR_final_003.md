## 2. AIエージェントの定義と技術構造

### 基本定義
AIエージェントは以下の4つの要素を満たすシステムと定義されます：

1. **目的**：目標を設定し、達成状況を評価する

3. **行動**：目標達成のためのタスクを実行する
4. **学習**：結果に基づいて行動を改善する

## 9. 投資対効果（ROI）の算出手法

### 効果の構造化
```
工数削減費 = 外部委託削減費 + 社内人件費削減費
           = (削減工数 × 外注単価) + (削減工数 × 人件費単価)
```

### 導入コストの全体像
```
導入コスト = 開発コスト + 運用保守コスト + 間接コスト
```

> **重要**：開発コストは氷山の一角。運用保守コストと間接コスト（教育、ルール整備等）が同規模になることも珍しくない。

### ROI評価のポイント
- 初期のROIがマイナスでも、横展開により部門全体でプラスに転じる可能性
- 単一業務のROIではなく、拡張性を含めた包括的評価が重要
- 実例：初年度ROI 0.8が、横展開により部門全体でROI 1.5に改善
