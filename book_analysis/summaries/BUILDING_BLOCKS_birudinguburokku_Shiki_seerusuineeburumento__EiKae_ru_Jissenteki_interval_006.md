- **コーチングとトレーニングの違い**: 一方的な指示や命令ではなく、質問を通じて営業担当者自身に事実を発見させるサポート役に徹する。

### コーチングの具体的ステップ
1. **導入準備**: セッションの目的、計画、価値、タイミングをすり合わせる。
2. **役割と意図の確認**: 担当者の自己成長を支援し、可能性を最大化する意図を伝え、信頼関係を築く。
3. **パフォーマンス分析**: 営業指標やコンピテンシーを一緒に診断し、対処すべき課題（コンピテンシーの不足など）の方向性を一致させる。
4. **ソリューションの検討と合意**: 複数のオプションから最適なソリューションを選択する。
5. **現場でのスキル育成**: 現場トレーニングを実施し、行き詰まった際の行動指針を明確にする。
6. **継続的な推進**: 従業員のモチベーション要因に配慮し、PDCAサイクルを回す。

> **重要**: アクションプランには「誰が、何を、どのように、いつまでに行うか」を明確に定義する必要がある。

## 営業プロセス（Sales Process）

### 営業プロセスの定義と構成要素
- **概要**: 営業活動が進む段階を意味する。複雑な用語を覚えるより、単純な概念を理解し目的を持って管理することが重要。
- **各段階の構成要素**:
  1. **目標**: 各段階で達成すべきこと。
  2. **タスク**: 目標を達成するための作業。
  3. **完了条件 (`Exit criteria`)**: 次の段階に移行するために満たすべき条件。
  4. **開始条件 (`Entry criteria`)**: その段階に入るための前提条件。

### 営業プロセスと営業メソッドの違い
- **営業プロセス**: 顧客の購買プロセスと連動する「ステージと完了条件の地図」。作業指示書（SOW）や相互アクションプラン（MAP）の作成に似ている。
- **営業メソッド**: 各段階で実行する「具体的なやり方（How-to）」。フレームワーク、モデル、スキル、ステップを含む。
- **顧客中心主義**: 担当者は、質問やヒントから顧客の現在の購買段階を特定し、各ステップの完了条件を満たしながら前進し続ける必要がある。
- **HAM/BAM**: `Have a Meeting, Book a Meeting`（会議が終わる前に次の会議を予約する）というテクニックが推奨される。

## 顧客ライフサイクルとアプローチ

営業プロセスは購買プロセスだけでなく、より広範な「顧客ライフサイブサイクル」と連動して設計される必要がある。

### ライフサイクルの各段階と有効な営業アプローチ
- **第1段階：事業計画の立案**
  - 顧客はまだサポートの必要性に気づいていない。
  - 有効な手法：**インサイトセリング**（顧客が気づいていない課題を指摘）、**目標ベースの営業（アウトカムセリング）**（データや専門性を共有し目標達成をサポート）。
- **第2段階：事業計画の実行**
  - 初期段階では顧客は自社内で問題解決を図る。
  - 有効な手法：**オブジェクティブベースセリング**。
- **第3段階：購買プロセス**
  - 顧客が外部の解決策を探し始める。
  - 有効な手法：**プロブレムベースセリング（問題解決型）**。他社が既に介入している場合は競争が激化するため、コンサルティング型で価値と成果を重視したアプローチが求められる。
- **第4段階：ソリューション実行（導入・評価）**
  - 競合他社のソリューションを置き換えるのは非常に困難。アプローチのタイミングの早さが重要。

> **注意**: プロスペクティング（見込み客発掘）は、顧客の変化と連動しない固有の段階として購買プロセスとは切り離して管理すべきである。
